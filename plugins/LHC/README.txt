This plugin is a reimplementation of the RSS one, but solely for reading LHC comments from the LHCb logbook

In order to use this plugin you must have the following modules
installed:
- feedparser: http://feedparser.org/

Basic usage
-----------

Add announcements for the feed
@LHC announce add

Stop announcements for the feed
@LHC announce remove

Get info about active feed
@LHC announce list

Get the last item 
@LHC last
